# docker-dubbo-compose

## Change Dir
> 改变磁盘目录到docker-compose下

``` bash
cd ./docker-compose
```

## Run
> 构建，（重新）创建，启动，链接一个服务相关的容器。

``` bash
docker-compose up
```
## start
> 启动一个已经存在的服务容器。

``` bash
docker-compose start
```
## Stop
>停止一个已经运行的容器，但不删除它。通过 docker-compose start 可以再次启动这些容器。

``` bash
docker-compose ps
```

## View
>列出所有容器

``` bash
docker-compose ps
```

## logs
>查看服务的输出。

``` bash
docker-compose logs
```
